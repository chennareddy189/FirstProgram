# FirstProgram
First Program example

# Building and running

This repository was converted to a Spring Boot Maven project.

Build:

```
cd D:\JenkinsPractice\FirstProgram
mvn clean package
```

Run (jar):

```
java -jar target\FirstProgram-0.0.1-SNAPSHOT.jar
```

Or run with Maven:

```
mvn spring-boot:run
```

Visit http://localhost:8080/ to see the greeting.
